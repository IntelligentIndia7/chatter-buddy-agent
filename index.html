Customer Support Bot Simulator

This is a Python-based Streamlit application that simulates a customer support interaction.

To run this application:

  1. Create a .env file with your Groq API key
  2. Install required packages: pip install -r requirements.txt
  3. Run the application: python run.py

This application is built with Python, Streamlit, LangChain, and LangGraph.

This HTML file serves as a landing page, but the actual application runs through Streamlit.
